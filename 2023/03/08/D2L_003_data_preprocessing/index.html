03_data_processing

data preprocessing

1) create a manul dataset

Manually input all the values of each rows and colmuns ,then save it in csv(Comma-separated values).

and we usually use pandas to load the csv file, so do some manipulations.

2) clear the lost data

typical methods to address the lost datas are

  1. insert numbers with estimates values (imputation )
  2. discards the whole rows or columns that contain missing values(deletion)
insert

using iloc,with indicate the explict location of the elements in tables to separate the specific element.And the code below presents how we can divide the table in to input (features of the rooms) and the output(price of the rooms)

get_dummies

Convert categorical variable into dummy/indicator variables.
just like a simple truth table —> use the form of truth table to visilize whether the item is NaN or not.

fillna

fillna allows us to fill the whole NaN number with stuffs,one of the choice is mean of the row.
once we point out the locations of all the NaN, we can eazily fill them with the mean of the input values.

3) Conversion to the Tensor Format

Now we clear all the lost values,and it’s time to convert the table into a tensor !

4) Discussion & Summary

prospection
1. dataset can be larger 
2. data visualization with tools like seaborn or matplotlib,etc.
3. etc.
summary

So when we get the original dataset, regardless of it’s size,what we should to do is obeying the following steps.

  1. separate the input values and the output values .In general , the output is what we want to predict.
  2. examniate the lost datas, point out the NaN or other bad datas with clear form to present.
  3. replace (or deplete) the bad data.
  4. convert the processed data into tensor form.
