Imagin a picture with 12M pixels, and each pixels use R,G,B ,three color channels, add up to a total 36M pixels.
If we input all of the pixels into the input layers of a MLP, and assum each of the pixel involveing 100 featrues, we ask at least 3.6B hidden units.
The number is terrible,we can’t implement that obviously.
So we need to find a way to address the problem.To begin with ,we need to extract some featrues when we process a pictrue.

19.2 Two principle

19.2.1 Overview

Two important principle to implement CNN are translation invariance and locality.
Always remember that we want to classify a picture, and find the pattern of a kind of pictures.
So firstly, the existence of some items in the picture should have no relationship with it’s location.
And for locality, we can just focus on the item we want to find, ignore other things which are far away .
